# Chawapol Tang | Sustainability Analyst Portfolio

This is a modern, multi-page portfolio website for **Chawapol Tang**, a Sustainability Analyst. The site is designed to showcase expertise in sustainability through a clean "modern white and rich green" theme.

## 🌿 Project Vision
To provide a professional platform highlighting data-driven insights and strategic ESG reporting aimed at building a greener, more resilient future.

## 🚀 Features
- **Modern Design:** Responsive layout with a professional color palette.
- **Multi-page Structure:**
    - `index.html`: Home page with a hero introduction and mission statement.
    - `projects.html`: Overview of core sustainability projects (LCA, Decarbonization, ESG Performance).
    - `contact.html`: Professional contact information and links.
- **Interactive Elements:**
    - Dark mode toggle for accessibility and user preference.
    - Responsive navigation bar.
- **Tech Stack:** HTML5, Vanilla CSS3, and JavaScript.
